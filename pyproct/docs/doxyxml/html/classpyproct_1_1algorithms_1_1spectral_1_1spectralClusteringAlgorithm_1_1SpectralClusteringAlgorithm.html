Detailed Description

Implementation of Normalized Spectral clustering with Lrw Laplacian (Shi and Malik 2000).

It tries to be both fast and memory efficient in order to be able to work with very big datasets.

Constructor.

Calculates the eigenvectors given a dataset distance matrix. The eigenvector distances would be the common part for clusterings with different k.

Parameters:
condensed_matrix,:The distance matrix of the dataset.
sigma_sq,:The squared value of sigma for the adjacency matrix calculation. If None, the value will be automatically calculated.
max_clusters,:Maximum number of clusters we will try with this algorithm (for instance with max_clusters = 10 we can try with ks in range [1..10]
laplacian_calculation_type,:The type of calculation.
store_W,:If True the object stores the adjacency matrix. Useful for testing.
verbose,:If True some messages will be printed.
