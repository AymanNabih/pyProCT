Detailed Description

Implementation of the Calinski-Harabasz (VRC) index, as described in T.

Calinski and J.Harabasz paper in 'Communications in Statistics 3(1), 1-27 (1974)'. Contains the necessary functions to calculate the index in the two forms presented.
