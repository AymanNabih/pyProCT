def pyproct.tools.plotTools.ballGraph (   number_of_balls,
  size,
  max_box_diameter,
  h_ball_separation,

  alphas,

It draws an image with N spheres (where N is typically the number of clusters we have).

This spheres are outlined and in the case they're representing a 'M' type cluster, they have an inner circle representing the amount of elements of 'A' and 'B' that it has.

Parameters:
number_of_balls,:The number of data analysis we have done (usually number of clusters).
size,:Size of the image to hold the ball graph.
max_box_diameter,:Is the diameter of the biggest possible sphere.
h_ball_separation,:Separation of each sphere in the x axis.
v_ball_separation,:Separation of each sphere in the y axis (usually enough to fit the 'data card')
sizes,:List containing the size of each cluster.
alphas,:Is a list containing the dispersion for each cluster.
colors,:Dictionary of colors for each type ('A','B' and 'M') with RGB color in tuple form (f.ex. (255,255,255) is white)
cluster_types,:List containing the type of each cluster ('A','B' or 'M' (Mixed))
A_sizes,:Total number of elements in 'A' clusters, f.ex. [2,3,0,4,5] In this case the third cluster would be of type B.
B_sizes,:The same, for 'B' type clusters.
Returns:
: The PIL image with the drawing and the maximum calculated diameter of the biggest sphere.

def pyproct.tools.plotTools.pieChartCreation (   graph_size,
  fracs,
  name1,
  name2,
  colors 
)

Creates the big pie chart of the report.

In this pie chart one fraction represents the amount of space sampled together by both trajectories, and the other two fractions represent the amount of space that was sampled by either A or B.

Parameters:
graph_size,:The graph size in pixels (?)
fracs,:a list or tuple containing the total number of elements of pure A, pure B, and mixed clusters.
name1,:String identifying the first trajectory.
name2,:String identifying the second trajectory.
colors,:dictionary with color descriptions for "A", "B" and "M" (Mixed) written in string format (for example "#FFFFFF" for white)
Returns:
: A PIL image of the pie chart.
