def pyproct.clustering.comparison.distrprob.kullbackLieblerDivergence.KullbackLeiblerDivergence.get_matrix_data (   cls,
  matrix,
  initial_element,
  number_of_elements 
)

A matrix generated from the concatenation of two pdbs may have 3 submatrices.

First is the pairwise matrix of the first pdb, second the pairwise matrix of the second, and third is the distance matrix of pdb1 vs pdb2 (in which info is duplicated, as its itself a pairwise distance matrix). This function grabs the data of one of the two first submatrices.

Parameters:
matrix,:The matrix we are talking about :/
initial_element,:is the index of the initial element of the pdb we want to extract the data. For instance if we are working with 2 trajectories of 3 and 4 frames, indexes are [tr1:[0,1,2] tr2:[3,4,5,6]], so to extract the first submatrix data, initial_element would be 0 and number_of_elements 3. Extracting the second submatrix will need a initial_element value of 3 and a number_of_elements value of 4.
number_of_elements,:As explained above, the number of models the pdb we are working with has.
Returns:
: A 1D numpy.array containing the submatrix data.
