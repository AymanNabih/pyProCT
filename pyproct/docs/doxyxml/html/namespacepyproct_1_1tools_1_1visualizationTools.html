def pyproct.tools.visualizationTools.calculate_bounding_box (   coordinates,
  backbone_trace = []

Calculates the bounding box that encloses all the atoms of two arrays.

Parameters:
coordinates,:An array containing coordinates for a set of conformations (3D array where dim 0 is the conformation, dim 1 is the atom and dim 2 the coordinate)
backbone_trace,:If used (and if it has length > 0) is a 2D array where dim 0 is the atom and dim 1 the coordinate.
Returns:
: A 3-tuple containing:
  • The bounding box corners.
  • The center of the b.b. .
  • The corner with maximum coordinates values.
