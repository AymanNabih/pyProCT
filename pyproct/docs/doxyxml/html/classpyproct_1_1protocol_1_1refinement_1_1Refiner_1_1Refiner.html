  initial_cluster,
  new_cluster 
)

It renames the elements of a cluster using a map array.For instance, if our dataset has 5 elements and the cluster to remap contains element 1, 3 and 4, then the elements map will remap 0->1, 1->3 and 2->4.

This mapping is implemented as an array so map[0] = 1, map[1] = 3 and map[2] = 4.
